Calculadora en Kotlin

Una calculadora moderna desarrollada en Kotlin para Android. Esta aplicación implementa todas las operaciones matemáticas básicas con una interfaz limpia y amigable. El proyecto demuestra el uso de las mejores prácticas de desarrollo en Android y la implementación de una arquitectura MVVM.

Características principales:

  • Interfaz de usuario moderna y responsive
  • Operaciones matemáticas básicas y avanzadas
  • Historial de operaciones
  • Modo oscuro/claro
